# Navachetana Livelihoods

**Empowering Growth Through Microfinance**

Official website for Navachetana Livelihoods Private Limited (NLPL) - a microfinance company dedicated to empowering rural communities, women entrepreneurs, and small businesses through accessible financial solutions.

## Features

- Responsive design (works on mobile, tablet, desktop)
- Modern UI with smooth animations
- SEO optimized with Open Graph tags
- Interactive FAQ section
- Contact form
- WhatsApp integration

## Pages & Sections

- Home (Hero section with trust badges)
- About Us
- Services (JLG Loans, Individual Loans, Agricultural Finance, Financial Literacy)
- How It Works (4-step process)
- Testimonials
- FAQ
- Contact

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Inter)

## Setup

1. Clone this repository
2. Add your logo files:
   - `Official_Logo-removebg-preview.png` (transparent background)
   - `Official Logo.jpg` (white background)
3. Update contact information in `index.html`
4. Deploy to GitHub Pages or any web server

## Customization

Update these items in `index.html`:
- Contact address, phone, email
- WhatsApp number (search for `wa.me`)
- Social media links
- Team member details
- Testimonials with real stories
- Statistics (Villages, Borrowers, etc.)
